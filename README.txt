Pizzaria Padrinos

Descrição do Projeto:
O Pizzaria Padrinos é um sistema web desenvolvido em Java para gerenciamento de pedidos em uma pizzaria,ele permite que os usuários realizem operações básicas de CRUD (Criar, Ler, Atualizar e Excluir) em pedidos, utilizando um banco de dados MySQL.
O sistema possui uma interface estilizada com JSP e CSS.

Funcionalidades
Incluir Pedido: Adicionar novos pedidos com nome do cliente, descrição e valor.
Listar Pedidos: Visualizar todos os pedidos realizados.
Editar Pedido: Atualizar as informações de um pedido existente.
Excluir Pedido: Remover pedidos do sistema.

Tecnologias Utilizadas
Java: Linguagem de programação principal do sistema.
JSP (JavaServer Pages): Para construção da interface web.
CSS: Para estilização da interface.
Servlets: Para controle das requisições e respostas HTTP.
DAO (Data Access Object): Para interagir com o banco de dados.
MySQL: Banco de dados relacional para armazenamento de informações.
Apache Tomcat: Servidor de aplicação.

Como Configurar o Projeto
Pré-requisitos
